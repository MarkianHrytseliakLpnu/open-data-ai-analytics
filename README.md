# Лабораторна робота №1

---

**Тема:** Використання системи контролю версій Git

**Мета:** Отримати навички роботи з системою контролю версій git та віддаленим сервером github, навчитися виконувати розробку в git/github.

Дані для роботи було взято з порталу відкритих даних https://data.gov.ua/

---

## Про датасет

Дані взято з державного порталу наборів даних https://data.gov.ua/

Інформація про отримане майно (обладнання, програмне забезпечення) у рамках міжнародної технічної допомоги
https://data.gov.ua/dataset/scsu-information-about-received-property

---

## Гіпотези щодо даних

1. Які проєкти (projectTitle) отримали найбільшу сумарну вартість активів, і чи існує значна диспропорція ("перекіс") у фінансуванні певних сфер (наприклад, кібербезпека vs кордон)?
2. Чи є на ринку донорів домінуючі організації (implementerName), які забезпечують левову частку поставок, чи допомога диверсифікована між багатьма виконавцями?
3. Як змінювалася інтенсивність надання допомоги (кількість передач або їх вартість) у часі? Чи є пікові періоди (місяці), коли надходило найбільше обладнання?
